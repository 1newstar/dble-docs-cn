# 3.1.X View Syntax
##  Syntax
###create view : 
                 CREATE [OR REPLACE] VIEW

                                view_name [(column_list)]

                                AS select_statement

###alert view :
                  ALERT  VIEW

                                view_name [(column_list)]

                                AS select_statement

###drop view:
                DROP VIEW [IF EXISTS] view_name [, view_name]

###show create view :
               SHOW CREATE VIEW view_name;
