###  3.4.1 Lock&unlock

####  3.4.1.1  Syntax
LOCK TABLES   tbl_name  <b>lock_type</b>

<b>lock_type</b>:  READ |  WRITE

<br/>

UNLOCK TABLES

####  3.4.1.2 举例

```
lock tables test_table read;
unlock tables;

```

####  3.4.1.3 限制

由于采用的druid的SQL解析器，存在issue https://github.com/alibaba/druid/issues/2000
