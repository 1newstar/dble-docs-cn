## 3.4.4 SET TRANSACTION Syntax
SET SESSION TRANSACTION ISOLATION LEVEL level   

**level**:  
     REPEATABLE READ  
   | READ COMMITTED  
   | READ UNCOMMITTED  
   | SERIALIZABLE  
   
SET @@SESSION.TX_ISOLATION = 'level_str' 

**level_str**:
     REPEATABLE-READ  
   | READ-COMMITTED  
   | READ-UNCOMMITTED  
   | SERIALIZABLE  
   
注：因为隔离级别不加session关键字语义不同，暂不支持
