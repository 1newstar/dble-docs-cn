### 3.2.4 UPDATE

#### 3.2.4.1 Syntax 

UPDATE **table_reference**

SET **col_name1**={**expr1**} [, **col_name2**={**expr2**}] ...

[WHERE **where_condition**]
 
#### 3.2.4.2 举例

```
UPDATE  test SET VALUE =1 where id=5;
```

#### 3.2.4.3 限制

* UPDATE语句中的where_condition部分只允许出现简单的条件，不能支持计算表达式以及子查询  
* 不支持多表Join 的UPDATE
